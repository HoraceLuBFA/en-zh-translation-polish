# 文本分析 → 归化尺度 → 准确性质检（策略层）

基于《高级英汉翻译理论与实践 第4版》（叶子南）第 11、12、13、15、16 章。
这一层决定“这篇文本该往中文靠拢多少”——是前两份候选 (techniques / symptoms) 的总开关：**先判定文本性质，再决定下手多狠**。

---

## 小节 A — 翻译前的文本分析：7 个维度 → 自由度

> 我们在翻译任何一篇文章时，都必须先弄清楚，这是一篇什么样的文章？……只有搞清楚了这些问题，我们才能回答……到底应该直译还是意译？到底应该更灵活些还是较死板些？（第15章）

落笔前过一遍这 7 个判定问句，给文本一个“自由度”打分（0=必须贴原文，10=可大改）：

1. **逻辑驱动 vs 情感驱动** (logic- vs emotion-driven)：经济/法律/科技/哲学=逻辑驱动，贴原文逻辑；抒情/闲谈/感叹=情感驱动，可灵活，一原文可对多译文。
2. **纽马克三类** （A 表达 / B 信息 / C 呼唤）：见小节 A2。
3. **正规 vs 非正规** （语域 register）：必须传达“语域的潜台词”——原文正式则译文正式，原文土话则译文土话，原文骂人则译文骂人。
4. **评价性强弱** (evaluative or not)：科技报道/法律=无评价→译者少介入、重准确；社论/广告/旅游手册=强评价→可大幅调整结构、重流畅。
5. **文化含金量高低** (culture-bound)：
   > 一个文本与某一特定文化关系越不紧密，翻译中就越不必调整其语言结构；相反……关系越紧密，翻译中结构调整的空间就越大。（第15章，引 Hatim & Mason）
6. **聚焦作者 / 情境 / 读者** (focus on writer/situation/reader)。
7. **软文本 vs 硬文本** （见小节 A3，最接地气的总判据）。

> 最后，文本分析仅提供文本特征，并不提供文本外的信息，而实际翻译却必须考虑文本外因素，特别是翻译目的。（第15章）

**判定结论 → 自由度 (1–10)**：商业合同≈2–3，经济学教科书≈4–6，游记≈7–8。自由度越低，techniques/symptoms 里的“大改”手段越要克制。

### A2 — 纽马克三类文本 → 工夫下在哪一层 (XYZ)

> A 表达功能 (expressive)……以作者为中心，如文学；B 信息功能 (informative)……十分客观，如科技报告；C 呼唤功能 (vocative)……以读者为中心，旨在影响读者……如广告或政治宣传。（第12章）

纽马克 XYZ 系数：Y=文字本身，X=现实所指，Z=读者脑中图像。

| 文本类 | 典型 | 工夫下在 | 策略 | 自由度 |
|---|---|---|---|---|
| **A 表达型** | 文学、权威讲话 | **Y （语言本身）** | 语义翻译；短句译短句（海明威），保留作者标记 | 低 |
| **B 信息型** | 科技、报告、合同 | **X （现实/逻辑）** | 交际翻译，贴逻辑；术语用固定译法 | 低 |
| **C 呼唤型** | 广告、宣传 | **Z （读者效果）** | 大胆交际翻译，可夸张、可换形象（"This is the car"→“举世无双”）| 高 |

> 由于英语和德语或法语之间的差异远小于英语和汉语之间的差异……纽马克的基本模式虽然对英汉翻译有所启发，但……英汉翻译中交流（交际）翻译法仍然是最主要的方法。（第12章）

→ **英译汉的默认档位是“交际翻译 / 向中文靠拢”**，只有 A 类权威表达型文本才上调到语义翻译、多照应 Y。

### A3 — 文本质地软硬（第4版新增，最实用的总开关）

> “硬”文本翻译时要非常注意原文逻辑思路的展开，不宜随便打破原文逻辑，选词要精准，译文不要突显个人风格……灵活度相对较小。而“软”文本……跳脱原文的机会较多，灵活度相对较大。（第15章）

| 硬文本（≈自由度 1–3）| 软文本（≈自由度 7–9）|
|---|---|
| 词义边缘清晰、解释余地小 | 词义边缘模糊、解释余地大 |
| 文本个体特征弱、译者不宜介入 | 文本个体特征强、译者常可介入 |
| 句法逻辑严密、标准常态语法 | 逻辑非其特征、可有非常态语法 |
| 本土文化含量低、功利实用 | 本土文化含量高、常无功利目的 |
| **政治/法律/经贸/科技** | **诗歌/散文/杂文/评论/讲演/宣传/广告** |

> 假如译者被锁在……“牢房”中，硬文本译者也许有一平方米的自由空间，软文本的译者就会有三平方米的自由空间，而一个好的译者会尽可能利用这个空间。（第15章）

→ 实操：**先把文本判成偏硬/偏软，再决定 symptoms 清单里哪些“病”要狠改、哪些可放过**。硬文本里某些“西化”是为精确性付的合理代价（异化可原谅，底线是基本通顺）；软文本里则尽量发挥意合、追求地道。

## 小节 B — 隐喻翻译决策

认知隐喻视角：先分清**概念隐喻**（思维基础）和**语言隐喻**（表达），再决定保留/糅合/舍弃。

> 那些奇特的隐喻有时不太好理解，因为它们依靠的基础并不是我们非常熟悉的……但认知隐喻的理论却认为，很多隐喻在跨文化的过程中可能并不那么难理解，因为它们背后的概念隐喻都是基于人的基本感知。（第13章）

**决策树**:

1. **基本隐喻 / 图像图式隐喻**（基于大小、上下、冷热、视觉、旅程等人类共通感知）→ **可直译保留**，英汉共享，无障碍。
   例：LOVE IS A JOURNEY → “分道扬镳”“走到十字路口”“进入死胡同”“原地踏步”。
2. **可直译 ≠ 应直译** → 还要看文本类型与目的：
   > 翻译并不是为了求得认知层面的对等，而是要求得语用层面的对等。因此有时见到隐喻仍需要避免原文的隐喻，用目标语言中更适合的表达法。（第13章）
   例：婚姻 "gone off the track" 直译“出轨”会和“有外遇”混淆 → 改译“脱离正轨”。
3. **文化专属隐喻**（如“卢沟桥的狮子”）→ 跨文化障碍大，**糅合或舍弃**，转为直述。
   例：“新闻系卧虎藏龙”→ 舍弃隐喻："the journalism department was full of highly talented people"（反向同理：英→汉时文化专属隐喻也常需落地）。
4. **用隐喻图像反推词义**（理解工具）：译文里某个抽象词（如 tensions）语义模糊时，恢复其概念隐喻图像 → 选出更准的译法（tensions=被两种力左右拉扯→“左右为难的困境”）。

## 小节 C — 准确性质检清单（润色最后一道关）

第16章专治“基本意思无大错、但细节欠准”的译文——这正是润色阶段要抓的。逐条核查：

- [ ] **未被逼到偏离却擅自改结构**：能按原结构译就别“舍近求远”。
  > 译者并没有被逼到硬要偏离原文的地步，直接按照原来的结构完全可以翻译……根本没有必要“舍近求远”。（第16章）
- [ ] **句型转换造成重心/焦点偏移**:`The next four years…turned out…` 把主语“四年”降成时间状语→重心从“时间”偏到“人”；`A is B` 擅自译成 `B is A` 会移焦点。硬文本里慎用。
- [ ] **漏译细节虚词/情态**:`do know` 的强调语气、`turned out` 的“结果竟是”、`ranks as` 的“位列”——严肃文本里不能丢。
- [ ] **擅自添词**：原文无“良好”“社会”等却加进去；意译过度=添油加醋。
- [ ] **搭配是否地道**：核查动宾、定中搭配（如“扎根于……身上”不当）。
- [ ] **指代 / 定语 vs 状语**：理清 it/they 指向，别把定语错当状语。
- [ ] **逻辑关系是否被改**：因果勿译成顺接，让步/条件勿丢。
- [ ] **专名 / 术语 / 数字**：人名地名查译音表统一，硬文本术语用固定译法，数字逐一核对。
- [ ] **语域是否对齐**：正式↔正式，口语↔口语，粗话↔粗话。
- [ ] **准确性 vs 流畅性的权衡**：硬文本/正规文本=准确优先（甚至准确>流畅）；软文本/非正规=可为流畅适度放松准确——
  > 这种对准确性要求的放松和对原文的理解错误是完全不同的，后者没有原谅的余地。（第15章）

> ⚠️ 关键红线：放松准确性 ≠ 理解错误。理解错误**没有原谅的余地**。
